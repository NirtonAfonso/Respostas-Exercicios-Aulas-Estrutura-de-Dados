# Turma 42CMP01M3A - Estrutura de Dados

Este espaco reune os materiais, exemplos e respostas de exercicios da turma **42CMP01M3A** na disciplina de **Estrutura de Dados** do Centro Universitario Jorge Amado.

Use esta branch como referencia da sua turma. Os arquivos estao organizados por aula, atividade ou revisao, conforme o conteudo trabalhado durante o semestre.

## Como consultar

1. Abra a pasta correspondente ao conteudo que deseja revisar.
2. Leia os enunciados, exemplos e solucoes disponiveis nos arquivos `.c` e `.md`.
3. Compare as solucoes com o que foi apresentado em sala e use os exemplos como apoio para estudar sintaxe, logica e estruturas de dados.
