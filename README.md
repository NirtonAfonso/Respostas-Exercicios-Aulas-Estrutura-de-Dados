# Respostas dos Exercicios - Estrutura de Dados

Este repositorio centraliza o acesso as respostas, exemplos e materiais de apoio das aulas de **Estrutura de Dados** dos cursos de Computacao do Centro Universitario Jorge Amado.

A branch `main` funciona apenas como pagina inicial do repositorio. Os arquivos de cada turma ficam separados em branches proprias, para facilitar a consulta e evitar mistura de materiais entre turmas.

## Como usar este repositorio

1. Encontre sua turma na tabela acima.
2. Clique no link direto correspondente.
3. Dentro da branch da sua turma, navegue pelas pastas de aulas, exercicios, revisoes e atividades.
4. Consulte os arquivos `.c` e `.md` conforme o conteudo trabalhado em sala.

## Conteudos disponiveis nas branches das turmas

Os conteudos podem variar conforme o andamento de cada turma, mas as branches reunem materiais como:

- exercicios da Aula 2;
- exercicios da Aula 3;
- exemplos da Aula 4 sobre `structs` e ponteiros;
- exemplos da Aula 5 sobre vetores e listas sequenciais;
- respostas de atividades estruturadas;
- revisao para AV1.

## Observacoes importantes

- Use sempre a branch da sua turma para acompanhar os materiais corretos.
- A branch `main` nao armazena respostas ou codigos de aulas; ela serve apenas como indice de navegacao.
- Caso algum link nao abra diretamente no GitHub, verifique se voce esta autenticado e se a branch ainda existe no repositorio remoto.
- Os materiais podem ser atualizados ao longo do semestre conforme novas aulas e atividades forem disponibilizadas.
